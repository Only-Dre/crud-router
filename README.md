# Conteúdos da Atividade
---
- Projeto CRUD com React Router e integração via Fetch API.
- Backend simulado com JSON-Server (db.json).
- Métodos HTTP implementados: GET, POST, PUT, DELETE.
- Cumpre os requisitos de uma versão anterior de crud-router.
---
# Objetivos
O objetivo desta atividade é refatorar o projeto de CRUD (Create, Read, Update, Delete) com Router para integrar uma API REST simulada utilizando o JSON-Server como mockup de banco de dados. A comunicação com o backend deve ser realizada exclusivamente através da Fetch API nativa do JavaScript, implementando as operações CRUD completas usando os métodos HTTP GET, POST, PUT e DELETE.
